# GitHub-Challenge-API
API used to GET information from Github

FollowersController
  GET
  Route - localhost/api/Followers/{id}
  Return Type - JSON

RepoController
  GET
  Route - localhost/api/Repo/{id}
  Return Type - JSON
  
StargazerController
  GET
  Route - localhost/api/Stargazers/{id}/{repositoryName}
  Return Type - JSON
